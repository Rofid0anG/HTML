(1). Pengenalan HTML

  HTML adalah bahasa markup standar untuk membuat halaman Web.

(2). Apa itu HTML?

  1. HTML adalah singkatan dari Hyper Text Markup Lenguage.
  2. HTML adalah bahasa markup standar untuk membuat halaman Web.
  3. HTML menggambarkan struktur halaman Web.

(3). Dokumen HTML sederhana

  Contoh:

    <!DOCTYPE html>
    <html>
    <head>
    <title>Judul halaman Web</title>
    </head>
    <body>

    <h1>Judul pertama saya</h1>
    <p>Paragraf pertama saya</p>

    </body>
    </html>

  Penjelasan:

    1. Deklarasi <!DOCTYPE html> mendefinisikan bahwa dokumen ini adalah dokumen HTML5.
    2. Elemen <html> adalah elemen root dari halaman HTML.
    3. Elemen <head> berisi informasi meta tentang HTML.
    4. Elemen <title> menentukan judul untuk halaman HTML.
    5. Elemen <body> mendefinisikan tubuh dokumen.
    6. Elemen <h1> mendefinisikan judul.
    7. Elemen <p> mendefinisikan paragraf.
