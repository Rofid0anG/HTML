Pengenalan HTML

HTML adalah bahasa markup standar untuk membuat halaman Web.

Apa itu HTML?

  • HTML adalah singkatan dari Hyper Text Markup Lenguage.
  • HTML adalah bahasa markup standar untuk membuat halaman Web.
  • HTML menggambarkan struktur halaman Web.
